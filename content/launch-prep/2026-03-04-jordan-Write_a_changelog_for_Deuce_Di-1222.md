**Deuce Diary - v1.0.0 - Release Notes**

**Date:** October 26, 2023

**Overview:**

Deuce Diary v1.0.0 is the initial release of Deuce Diary, a personal diary application designed to help you document your thoughts, experiences, and observations. This version includes core functionality for creating and managing diary entries, and basic customization options.

**New Features:**

* **Diary Entry Creation:**  You can now create new diary entries with a title and a detailed text field for your thoughts.
* **Entry Storage:**  Entries are automatically saved to the device's storage and will persist between sessions.
* **Basic Formatting:**  Supports basic text formatting, including:
    * **Bold Text:**  Use `**text**` to make text bold.
    * **Italic Text:**  Use `*text*` to make text italic.
* **Entry Listing:**  A simple list view displays all created diary entries.
* **Entry Editing:**  Existing entries can be edited directly within the application.
* **Entry Deletion:**  Entries can be deleted using a swipe-to-delete gesture.
* **Dark Mode Support:** Includes a built-in Dark Mode for comfortable viewing in low-light conditions.  Switch between Light and Dark Mode using the settings menu.

**Bug Fixes:**

* **Crash on Entry Creation:** Fixed a crash that occurred when attempting to create a new entry with excessively long titles.
* **Incorrect Date Display:** Resolved an issue where the date format in some entries was incorrect.
* **Saving Issues:** Fixed a bug where entries were occasionally not being saved correctly.
* **UI Glitch:** Corrected a minor UI glitch in the entry list display.


**Known Issues / Limitations:**

* **No Search Functionality:**  Currently, there is no search functionality to find specific entries.
* **Limited Formatting Options:** The formatting options are basic and do not support other markdown elements.
* **No Import/Export:**  There is currently no support for importing or exporting diary entries.
* **Simple UI:** The user interface is intentionally simple for this initial release.

**Future Development (Planned for Future Releases):**

* Search Functionality
* Import/Export (e.g., Markdown, Text)
* Enhanced Formatting Options (e.g., Headers, Lists)
* Tagging and Categorization
* Cloud Synchronization (for backup and access across devices)



**Notes:**

* This is the first release of Deuce Diary. We appreciate your feedback and encourage you to report any issues or suggestions you may have.
* Thank you for trying Deuce Diary!
