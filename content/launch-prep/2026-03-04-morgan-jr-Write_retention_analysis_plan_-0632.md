**2. Data Collection & Definition**

* **User Definition:** Clearly define what constitutes a "user" for this analysis (e.g., any account created, active user, paying user).
* **Data Sources:**
    * **Product Analytics Platform:** (e.g., Mixpanel, Amplitude, Google Analytics with Enhanced Ecommerce, Heap) - This is the primary data source.

* **Key Metrics:**
    * **New User Retention Rate (Day 1, Day 7, Day 30):**  (Number of users still active) / (Total number of new users) * 100
    * **DAU/MAU (Daily Active Users / Monthly Active Users):**  A measure of engagement over time.  Track this across all time horizons.
    * **Churn Rate:** (1 - Retention Rate) * 100 -  Percentage of users who stop using the product.
    * **Activation Rate:** (Percentage of users who complete a key activation step) – This can indicate early engagement.
    * **Feature Usage:**  Track which features users are utilizing, and when they stop using them.
    * **Session Length/Frequency:**  How long and often users are engaging with the product.
    * **User Segmentation Data:** Demographic data, signup source, onboarding flow, etc.
