* **Color Contrast:** We have implemented sufficient color contrast between text and background elements to improve readability for users with low vision.  We are using tools to monitor and adjust contrast ratios.
* **Resizable Text:** Text on the website is scalable, allowing users to adjust the font size to their preference within their browser settings.
* **Semantic HTML:** We utilize semantic HTML elements (e.g., headings, lists, landmarks) to structure content logically, aiding screen reader users and improving overall website usability.
* **Form Accessibility:** Forms are designed to be accessible, with clearly labeled fields and instructions for completion.
