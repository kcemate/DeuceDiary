**Deuce Diary v1.0.0 - Launch Changelog**

**Release Date:** October 26, 2023

**Summary:**  Deuce Diary v1.0.0 is the official launch of the application, providing a central hub for users to track their habits, set goals, and engage in a supportive community. This release includes core functionality for habit tracking, social interaction, and a user-friendly interface.

**I. Social Features**

* **Community Feed:**
    * Users can now post updates, share their progress, and ask for support within a dedicated community feed.
    * Posts are visible to all Deuce Diary users.
    * Like and comment functionality on posts are implemented.
* **Following & Followers:**
    * Users can follow other users to see their activity and contributions in their feed.
    * A follower count is displayed on user profiles.
* **Direct Messaging (Beta):**
    *  A basic, private messaging system has been implemented (currently in beta and may be unstable). Users can send direct messages to followed users. (Limited to 10 messages per day)
* **Group Creation (Coming Soon - Planned for v1.1):**  The groundwork for group creation functionality has been laid, with initial database setup completed.

**II. Tracking Features**

* **Habit Creation:**
    * Users can create new habits with customizable names, descriptions, and target frequencies (daily, weekly, monthly).
    * Habit categories are supported (e.g., Health, Productivity, Personal Development).
* **Habit Logging:**
    * Simple “Mark as Complete” button for each habit.
    * Habit completion history is automatically recorded.
* **Habit Streak Tracking:**
    * The application tracks consecutive days/weeks/months a habit has been completed, displaying a streak.
* **Reminders:**
    * Users can set reminders for their habits, triggered by specified times or days. (Reminder notifications are currently web-based only)
* **Habit Scheduling:**
     * Users can set a specific time for their habits to occur daily, weekly, or monthly, with customizable options.
* **Habit Editing:**
    * Users can modify the parameters of existing habits (name, frequency, reminders).

**III. Gamification Features**

* **Points System:**
    * Users earn points for completing habits and contributing to the community (e.g., liking posts).
* **Leveling System:**
    * Points accumulate towards leveling up, unlocking cosmetic rewards (badges & profile customization options).
* **Badges:**
    * Earned for specific achievements: longest streak, active community member, etc. (Initial badge set included).
* **Profile Customization:**
    * Users can personalize their profiles with a chosen avatar, background, and status updates.

**IV. UI/UX Improvements**

* **Modern Design:**  A visually appealing and intuitive design has been implemented, focusing on ease of use.
* **Responsive Layout:** The application is designed to adapt to various screen sizes (desktop, tablet, mobile).
* **Dark Mode:** Users can toggle between light and dark mode.
* **Improved Navigation:** Simplified navigation structure for easier habit management and community access.
* **Interactive Habit Visualization:**  Visual representation of habit completion streaks and progress over time (graphs).


**V. Infrastructure & Technical**

* **Backend:**
    *  Developed using Node.js with Express.js framework.
    *  Database: MongoDB (for flexible data storage).
    *  Cloud Hosting: Deployed on AWS (Amazon Web Services).
