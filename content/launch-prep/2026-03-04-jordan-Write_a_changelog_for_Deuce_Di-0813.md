**Deuce Diary - Version 1.0.0 - Release Log**

**Release Date:** October 26, 2023

**Introduction:**

Deuce Diary v1.0.0 is the initial release of Deuce Diary, a personal journal application designed for thoughtful reflection and creative writing. This release provides the core functionality to create, edit, and view journal entries.

**New Features:**

* **Core Journaling Functionality:**
    * **Entry Creation:**  Ability to create new journal entries with a title and body text.
    * **Rich Text Editor:**  Basic formatting options (bold, italics) within journal entries.
    * **Date & Time Stamping:**  Each entry automatically records the date and time of creation.
* **Entry Management:**
    * **Entry Viewing:**  Entries are displayed in chronological order, with the most recent at the top.
    * **Entry Editing:**  Existing entries can be modified and saved.
    * **Entry Deletion:**  Entries can be permanently deleted. (Confirmation prompt included)
* **Basic UI/UX:**
    * **Clean and Minimalist Design:**  Focused on providing a distraction-free writing environment.
    * **Simple Navigation:**  Intuitive controls for creating, editing, and viewing entries.
* **Local Storage:** All journal entries are stored locally on your device.

**Bug Fixes:**

* **Fixed:** Resolved an issue where the application would crash upon creating a new entry with a very long title.
