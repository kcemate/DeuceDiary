**2. Data Requirements & Tracking:**

* **Data Sources:**

* **Key Metrics:**
    * **Daily Active Users (DAU):** Number of unique users active in a 24-hour period.
    * **Weekly Active Users (WAU):** Number of unique users active in a 7-day period.
    * **Monthly Active Users (MAU):** Number of unique users active in a 30-day period.
    * **Retention Rate:** Percentage of users who return after a specific period (e.g., % of users who logged in on D7 after logging in on D1).
    * **Churn Rate:** Percentage of users who stopped using the product/service during a specific period. (Calculated as 1 - Retention Rate)
    * **Cohort Analysis:** Group users based on signup date to track retention patterns over time.
    * **Time to Value (TTV):** The time it takes for a new user to experience the core value of the product. (A key predictor of early retention)
