**Deuce Diary - v1.0.0 - Changelog**

**Release Date:** October 26, 2023

**Overview:** This is the initial release of Deuce Diary, a personal journal application designed for tracking daily activities, thoughts, and moods within the context of the *Star Wars: Knights of the Old Republic* universe.

**New Features:**

* **Core Journal Entry Functionality:**
    * **Daily Entry Creation:**  You can now create and save new journal entries for each day.
    * **Text Input:**  A robust text editor allows you to write freely about your experiences.
    * **Date & Time Stamping:**  Each entry is automatically timestamped with the current date and time.
* **Character Profile Setup:**
    * **Character Creation:**  Start a new journal by creating a character profile. Choose from a selection of potential Jedi, Bounty Hunters, and other characters found in the game.
    * **Basic Stats:**  Initial character profiles include a name and basic background choice.
* **Mood Tracking:**
    * **Mood Selection:**  You can now assign a mood (e.g., "Hopeful," "Wary," "Angry") to each journal entry, providing a visual representation of your emotional state throughout your journey.
* **Basic UI:**  A simple, intuitive user interface designed to facilitate daily journal writing.
* **Saving & Loading:**  Entries are automatically saved and can be loaded for continued writing.


**Bug Fixes:**

* **Loading Errors:** Resolved an issue where the application would crash upon loading older journal entries.
* **UI Alignment Issues:**  Fixed minor UI alignment problems that occurred on some screen resolutions.
* **Date Formatting:** Corrected a problem where dates were being displayed in an incorrect format.
* **Empty Entry Handling:**  Improved handling of empty journal entries to prevent potential errors.


**Known Issues & Limitations:**

* **No Dark Mode:**  Currently, the application only supports light mode.
* **Limited Character Options:**  The initial character selection is limited.  Future versions will expand this significantly.
* **Basic Mood Options:** The mood selection is currently quite basic.  We plan to add more granular emotion options.
* **No Import/Export:** The application doesn't currently support importing or exporting journal entries (we plan to add this feature in a future release).
* **No Spellcasting Tracking:** The app does not currently track spells or abilities used during adventures.
