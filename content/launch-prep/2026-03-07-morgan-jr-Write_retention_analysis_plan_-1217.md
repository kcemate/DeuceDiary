**1. Goal & Objectives:**

**2. Data Collection & Tracking:**

* **Key Metrics:**
    * **D1 Retention:** Percentage of users who return to the product/service one day after signing up.
    * **D7 Retention:** Percentage of users who return to the product/service seven days after signing up.
    * **D30 Retention:** Percentage of users who return to the product/service thirty days after signing up.

    * **Churn Rate:** The inverse of retention (100% - Retention).
    * **Feature Usage:** Track which features are used most frequently by retained users versus churned users.
    * **User Segmentation:** Segment users by key characteristics (e.g., acquisition channel, demographics, initial usage patterns).  This is *critical* for identifying targeted intervention opportunities.
* **Tracking Tools:**
    * **Analytics Platform:** Google Analytics, Mixpanel, Amplitude, Heap, or your preferred analytics solution.  Configure event tracking to capture user actions.
    * **CRM System:** (e.g., Salesforce, HubSpot)  To store user data and track interactions.
    * **Database:** For storing raw user data and intermediate analysis results.
* **Data Definition:**  Clearly define what constitutes a "returning user." This should be based on a consistent login or key action.
