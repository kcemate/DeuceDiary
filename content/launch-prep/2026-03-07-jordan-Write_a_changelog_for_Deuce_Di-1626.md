**Deuce Diary v1.0.0 – Release Notes**

**Date:** October 26, 2023

**Introduction:**

Deuce Diary is a brand new interactive experience exploring the world of a young, enigmatic entity named Deuce. This release represents the initial launch of the game and includes core gameplay features, a basic story, and early art assets.

**New Features:**

* **Core Gameplay Introduced:**  The central mechanic of “Tracking” Deuce’s movements is now fully implemented.  You can use your mouse to follow his indicated trail.
* **Basic Storyline:** The initial story sequence is playable, introducing players to Deuce’s situation and the core mystery. (Approximately 15-20 minutes of gameplay).
* **Interactive Environment:** You can now click on key objects within the initial environment to trigger minor events and uncover early clues.
* **UI Framework Established:** A basic user interface (UI) has been built, including a simple status display and the Tracking indicator.
* **Audio Foundation:**  The initial soundscape has been implemented, with ambient sounds and a short introductory music track.
* **Downloadable Content (Initial):** The game is immediately playable upon download.

**Bug Fixes:**

* **Tracking Drift:** Resolved a significant issue where the Tracking indicator would occasionally drift uncontrollably.  The tracking system is now far more stable.
* **UI Element Clipping:** Fixed an issue where a UI element would clip through the environment in specific locations.
* **Save Game Error (Initial):** Resolved a bug that prevented the save game from being correctly loaded on first launch.
* **Minor Audio Glitches:**  Identified and fixed several minor audio glitches throughout the initial level.

**Known Issues & Limitations:**

* **Limited Level Design:** The game currently only features a single, small level.

* **No Difficulty Options:** The game currently has no difficulty settings.
* **Text-Based Dialogue:** All dialogue is currently presented as text; we plan to implement voice acting in later versions.
