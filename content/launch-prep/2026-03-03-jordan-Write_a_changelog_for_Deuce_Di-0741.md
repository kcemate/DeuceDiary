Deuce Diary v1.0.0 is the initial release of Deuce Diary, a personal journal application designed for creative writing and thoughtful reflection. This version introduces core functionality for capturing and organizing your thoughts, allowing you to build your own digital diary.

* **Entry Listing:**  Entries are displayed in chronological order within a scrollable list.

* **Basic Search:** A simple search function allows you to quickly find entries by title.
* **Dark Mode:**  A dark mode option is now available for a more comfortable writing experience.
* **Initial Settings Screen:**  Allows you to select the preferred date format.
