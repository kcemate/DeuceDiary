**Deuce Diary - v1.0.0 - Release Notes**

**Release Date:** October 26, 2023

**Introduction:**

Deuce Diary v1.0.0 is the initial public release of Deuce Diary, a simple app designed for recording daily observations and thoughts, inspired by the work of Neal Stephenson's *Snow Crash* and the protagonist Deuce Bigalow.  This version provides a core set of features for logging your day, setting reminders, and tracking your progress.

**New Features:**

* **Daily Entry Creation:**  The core functionality - you can now create and save new daily entries.  Each entry includes a date, title, and a text field for your thoughts.
* **Reminder System:**  Set daily reminders to prompt yourself to record your thoughts. You can customize the reminder time.
* **Date Picker:**  Utilize a simple date picker to easily select the date for your entries.
* **Basic Text Editor:**  A text editor allows you to format your entries with basic Markdown syntax (bold, italics).
* **Data Persistence:** Entries are now saved locally to your device, ensuring your diary isn't lost when you close the app.
* **Clean & Minimal Interface:**  The app features a streamlined and intuitive interface, focusing on ease of use.


**Bug Fixes & Improvements:**

* Fixed an issue where the app would crash when creating a new entry with a very long title.
* Improved date handling to accurately display dates across different locales.
* Enhanced text editor rendering for better readability of Markdown formatting.
* Resolved a minor UI glitch related to the reminder icon’s positioning.
* Optimized app loading time for a smoother initial experience.

**Known Issues & Limitations:**

* **No Cloud Sync:** Currently, Deuce Diary does not support cloud synchronization. Your diary data is stored locally and will be lost if the app is uninstalled or your device is reset.  (A cloud sync feature is planned for future releases.)
* **Limited Formatting:** Markdown support is basic. More advanced formatting options are not yet implemented.
* **No Search Functionality:**  There is no built-in search feature to quickly find specific entries.
* **Basic Reminder Only:** Reminders are simple alerts and don't offer advanced features like snooze or custom sounds.
