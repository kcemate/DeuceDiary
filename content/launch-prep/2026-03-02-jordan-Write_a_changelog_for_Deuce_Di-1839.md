## Deuce Diary v1.0.0 Changelog - Launch!

**Release Date:** October 26, 2023

**Summary:** We're thrilled to announce the launch of Deuce Diary v1.0.0! This initial release focuses on core functionality for habit tracking and community support, laying the groundwork for a thriving community of self-improvement enthusiasts. This version introduces key features for tracking, social interaction, and a streamlined user experience.

**I. Social Features:**

* **Community Forum:** Introduced a basic community forum where users can share their progress, offer encouragement, and ask questions.
    *  *Functionality:* Browse threads, create new threads, reply to posts, upvote/downvote posts.
    *  *Note:*  Moderation features are currently limited and will be expanded in future updates.
* **Follow Feature:** Users can now follow other users to see their activity and progress (publicly).
    * *Functionality:*  Click “Follow” button on a user’s profile. Follower counts displayed on user profiles.
* **Daily Encouragement Posts:** A daily automated post within the community forum offering a motivational message.
* **Basic User Profiles:** Users can now create profiles with:
    *  *Functionality:*  Username, optional profile picture, a short bio, and a display of their tracked habits.



**II. Tracking Features:**

* **Habit Creation:** Easily add new habits with customizable names, descriptions, and frequencies.
    * *Functionality:* Set habit name, detailed description, choose frequency (Daily, Weekly, Specific Days), and set a target duration.
* **Habit Tracking:** Log completed and missed habits directly within the app.
    * *Functionality:* Simple toggle to mark habits as completed or missed.
* **Habit Categories:** Organize habits into customizable categories (e.g., Health, Productivity, Personal Development).
    * *Functionality:* Add, edit, and delete categories. Assign habits to specific categories.
* **Streak Tracking:** Automatic calculation and display of current habit streaks.
    * *Functionality:* Displays current streak for each habit.
* **Detailed Habit History:** View a chronological history of completed and missed habits for each habit.
    * *Functionality:*  Date and time stamps for each logged activity.
