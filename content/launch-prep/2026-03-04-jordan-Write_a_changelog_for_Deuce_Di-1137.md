**Deuce Diary - Version 1.0.0 - Changelog**

**Introduction:**

Deuce Diary is a personal journaling application designed for quick note-taking and tracking daily events. This release marks the initial public version of the application and includes core functionality for creating, editing, and viewing diary entries.

**New Features:**

* **Core Journal Entry Creation:** The ability to create new diary entries with a title, date, and body text.
* **Date Selection:** A date picker allows you to easily select the date for your entry. (Supports browsing by year, month, and day)
* **Text Editor:** A basic text editor within each entry allows for easy writing and formatting of your thoughts.
* **Entry Viewing:**  Entries are displayed chronologically in a list format.
* **Entry Editing:** Existing diary entries can be edited directly within the application.
* **Saving & Loading:** Entries are automatically saved locally, and the application can load previously saved entries.
* **Dark Mode:**  A dark mode theme is included, offering a comfortable viewing experience in low-light conditions.
* **Simple UI:** A clean and intuitive user interface designed for quick note-taking.
