# Bingo UI Iterations

## Iteration 1: Bingo Line Highlighting
**Problem:** No visual distinction when a row/col/diagonal is complete — users can't see their BINGO at a glance.
**Change:** Compute which squares belong to a completed line (row, col, diagonal). Highlight those squares with a green ring + deeper gold gradient to differentiate from "just completed" squares.
